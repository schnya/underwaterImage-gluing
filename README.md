# 環境構築・前準備
[pipenv](https://github.com/pypa/pipenv) で環境を作成します。pipenv の使い方は[公式ドキュメント](https://pipenv-ja.readthedocs.io/ja/translate-ja/)を参照してください。


```python
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```

# 実行
```
python3 main.py
```
